### Running scans

1. Quick view
   ```sh
   docker scout quickview <Docker Image name>
   ```
2. Get Recommendations from Scout
   ```sh
   docker scout recommendations <Docker Image name>
   ```
3. Vulnerability CVEs
   ```sh
   docker scout cves <Docker Image name>
   ```
